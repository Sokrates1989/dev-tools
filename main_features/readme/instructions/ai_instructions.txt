# Instructions for AI:

IMPORTANT:
- DO NOT use triple backticks ``` inside the generated markdown content.
- Instead, use triple semicolons ;;; for code blocks, e.g.

From this:
```bash
echo hello
```

To this:
;;;bash
echo hello
;;;

- The final output MUST be fully wrapped inside:

```markdown
YOUR OUTPUT GOES HERE
```

...but again: NEVER use ``` anywhere *inside* the markdown body! Use only ;;; instead.
